# Case Study: Figma — Community-Led PLG to Enterprise PLS

**Valuation:** $10B+
**GTM Motion:** [[Community-Led Growth]] → [[Product-Led Growth]] → [[Product-Led Sales]]
**Key Stat:** 95% organic traffic

## The 5-Phase Playbook

### Phase 1: Stealth Community Building (2015-2016)
- Built community **before the product was publicly available**
- Seeded design communities with early access
- Gathered feedback, created anticipation
- Philosophy: community first, product second

### Phase 2: Bottoms-Up Launch (2016-2019)
- Free, browser-based design tool with **real-time collaboration**
- **The viral loop:** One designer shares a Figma file via URL → multiple teams start collaborating
- No sales team needed — the product spread itself
- URL sharing = zero-friction onboarding (no download, no install, no account needed to view)

### Phase 3: Community Expansion (2019-2021)
- Notion Ambassador-style programs for local communities
- **Template marketplace** — users create and share design resources
- **Community files** — open-source design systems, UI kits
- **Plugins** — third-party ecosystem extending the product
- Result: **95% organic traffic** — almost no paid acquisition

### Phase 4: Product-Led Sales (2021-2023)
- Added **enterprise features**: SSO, admin controls, design system management
- Sales team engaged teams already spending **>$100/month** on self-serve
- Sales didn't replace PLG — it **augmented** it
- Enterprise deals built on top of existing bottoms-up adoption

### Phase 5: Multi-Product (2023+)
Sequential expansion of the buyer base:

| Product | Target User | Effect |
|---------|-------------|--------|
| **Figma Design** | Designers | Core product |
| **FigJam** | PMs, brainstorming | Expanded beyond designers |
| **Dev Mode** | Engineers | Expanded to entire product team |
| **Figma Slides** | Everyone | Further expansion |

Each product brought new users into the Figma ecosystem, creating cross-sell opportunities.

## The Viral Mechanics

Figma's viral loop is uniquely powerful because it's **embedded in the collaboration workflow**:

```
Designer creates file → Shares URL with PM/engineer → They view in browser
                                                        ↓
                                                   Start using Figma
                                                        ↓
                                                   Invite their team
                                                        ↓
                                                   Team hits spending threshold
                                                        ↓
                                                   Sales engages for enterprise
```

Unlike Dropbox ("invite a friend"), Figma's sharing is part of the **core workflow** — you can't design collaboratively without sharing.

## Key Numbers

- **95%** organic traffic
- **$10B+** valuation
- Config conference draws thousands of designers worldwide
- Template marketplace creates **compounding** content flywheel

## Key Lessons

1. **Community before product** — Build the audience, then build the product for them
2. **The product IS the GTM** — URL sharing as a viral loop requires zero marketing
3. **Don't fight enterprise sales — add it when ready** — Let bottoms-up adoption prove demand, then layer sales on top
4. **Multi-product expands the buyer** — Each new product = new user persona = wider organizational adoption
5. **Templates and plugins create defensibility** — User-generated content creates switching costs
