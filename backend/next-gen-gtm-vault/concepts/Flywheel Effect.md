# Flywheel Effect

**Jim Collins' flywheel concept** applied to GTM. Each turn of the flywheel makes the next turn easier, creating compounding, self-reinforcing momentum.

## The GTM Flywheel

```
        ┌──── More Data ────┐
        │                    │
 Better Outcomes      Better Models
        │                    │
 More Customers      Better Targeting
        │                    │
 More Signals        Better Timing
        │                    │
        └──── More Data ────┘
```

### How Each Turn Feeds the Next

1. **More data** (interactions, signals, outcomes) feeds...
2. **Better models** (scoring, ICP, intent, health) which enable...
3. **Better targeting** (right accounts, right contacts) which leads to...
4. **Better timing** (right moment, right channel) which produces...
5. **Better outcomes** (higher conversion, lower churn) which attract...
6. **More customers** who generate...
7. **More signals** (usage, engagement, expansion) which create...
8. → **More data** and the flywheel accelerates

## The Data Network Effect

Each new customer doesn't just add revenue — they add **data** that makes the system smarter for everyone:

- More deals → better win/loss patterns → better scoring
- More usage → better PQL models → better timing
- More outbound → better message optimization → better conversion
- More churn data → better health models → better retention

This is a **data network effect** — the system improves for all users as each interaction is added.

## Why the Flywheel Is a Moat

- You can copy tools (public)
- You can copy playbooks (shared)
- You **cannot copy 24 months of compound learning** (unique to your data + market + customers)

A competitor starting today is 24 months behind in model accuracy, signal calibration, and optimization history. That gap **widens** over time.

## Measuring the Flywheel

Track these to know if your flywheel is spinning:

| Metric | What It Shows | Target |
|--------|-------------|--------|
| Model accuracy trend | Are predictions getting better? | Improving monthly |
| Conversion rate trend | Is outbound getting more effective? | Improving monthly |
| Feedback loop latency | How fast from action → learning? | Decreasing |
| Experiment velocity | How many experiments per week? | Increasing |
| Data volume growth | Is the flywheel generating more data? | Growing |
