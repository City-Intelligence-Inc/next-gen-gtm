# MEDDIC / MEDDPICC

The gold standard for **enterprise deal qualification**. First adopted in the 1990s at PTC, evolved for modern revenue organizations. Used by Snowflake, MongoDB, Databricks, and most enterprise SaaS companies.

## The Framework

| Letter | MEDDIC | MEDDPICC Addition |
|--------|--------|-------------------|
| **M** | **Metrics** — Quantified business impact the customer expects | |
| **E** | **Economic Buyer** — Person with budget authority and veto power | |
| **D** | **Decision Criteria** — How the buyer evaluates and compares solutions | |
| **D** | **Decision Process** — Steps, timeline, and stakeholders to purchase | |
| **I** | **Identify Pain** — The critical business problem driving the purchase | |
| **C** | **Champion** — Internal advocate who sells for you when you're not in the room | |
| **P** | | **Paper Process** — Legal, procurement, security review steps |
| **C** | | **Competition** — Who else is being evaluated |

## How It's Used in Practice

### Tiered Approach
- **MEDDIC** for deals <$50K — simpler qualification, faster cycles
- **Full MEDDPICC** for strategic enterprise deals >$50K — every element must be mapped

### Deal Review Cadence
1. **Discovery call** — Start mapping M, I (Metrics, Identify Pain)
2. **Demo/eval** — Map D, D (Decision Criteria, Decision Process)
3. **Champion building** — Identify and coach C (Champion)
4. **Proposal stage** — Confirm E (Economic Buyer), map P (Paper Process)
5. **Negotiation** — Address C (Competition)

### Each Letter = A Qualification Gate

If you can't fill in a letter, the deal has a gap. Common failure modes:
- **No Champion** → Deal stalls after demo (nobody selling internally)
- **No Economic Buyer access** → "We need to check with leadership" surprise at close
- **No Metrics** → Can't justify ROI → budget gets cut
- **Unknown Paper Process** → Deal slips quarter after quarter

## How AI is Changing MEDDIC

1. **Automated call analysis** — Gong/Chorus extract MEDDIC elements from sales calls
2. **Deal scoring** — AI scores each MEDDIC element based on conversation data
3. **Gap identification** — AI surfaces which elements are missing per deal
4. **Champion detection** — NLP identifies who's advocating for you in email threads
5. **Competitive intelligence** — AI monitors competitor mentions in deals

## Modern Alternatives

| Framework | Focus | Best For |
|-----------|-------|----------|
| **MEDDPICC** | Comprehensive enterprise qualification | $50K+ deals, long cycles |
| **BANT** | Budget, Authority, Need, Timeline | Quick qualification, transactional |
| **SPICED** | Situation, Pain, Impact, Critical Event, Decision | Buyer-centric, modern |
| **SNAP** | Simple, iNvaluable, Aligned, Priority | Fast-moving, competitive markets |
