# Case Study: Datadog — Multi-Product Land-and-Expand

**Revenue Growth:** New logo bookings more than doubled YoY
**NRR:** ~120% net revenue retention
**GTM Motion:** Self-serve + segmented sales + multi-product expansion
**Key Stat:** 84% of customers use 2+ products

## The Playbook

### 1. Easy Adoption
- Products with **very short time-to-value**
- Self-service onboarding — install agent, see data in minutes
- Free tier to reduce friction

### 2. Land Small
- Start with one product: **infrastructure monitoring**
- Smallest customers go fully self-serve
- No sales involvement needed for initial adoption

### 3. Expand Relentlessly
The multi-product strategy is Datadog's superpower:

| Product | Launch | What It Does |
|---------|--------|-------------|
| Infrastructure Monitoring | Core | Server and cloud metrics |
| APM (Application Performance) | Expansion | Application-level tracing |
| Log Management | Expansion | Centralized logging |
| Security Monitoring | Expansion | Threat detection |
| RUM (Real User Monitoring) | Expansion | Frontend performance |
| CI Visibility | Expansion | CI/CD pipeline analytics |
| Database Monitoring | Expansion | Query performance |
| Network Monitoring | Expansion | Network flow analysis |

**Cross-sell metrics (Q3 2025):**
- **84%** of customers use 2+ products
- **54%** use 4+ products
- **31%** use 6+ products
- **16%** use 8+ products

### 4. Segment the GTM Motion

| Segment | ACV | Motion |
|---------|-----|--------|
| SMB | <$10K | Fully self-serve |
| Mid-market | $10K-$100K | Inside sales |
| Enterprise | $100K+ | Dedicated AE + SE |

### 5. Revenue Composition
- **25%** of revenue from new customers
- **75%** from expansion within existing accounts
- ~**120% NRR** — existing customers spend 20%+ more each year

## Why Multi-Product Works

```
Land with monitoring → Team sees value → Need logs too? Already trust Datadog
    ↓                                         ↓
 1 product                              2 products
                                              ↓
                                      Need APM? Security?
                                              ↓
                                        4+ products
                                              ↓
                                    Enterprise contract
```

Each new product:
1. Solves an adjacent problem the customer already has
2. Integrates natively with existing Datadog products
3. Is easier to adopt than a new vendor
4. Increases switching costs

## Key Lessons

1. **Land small, expand big** — Don't try to sell everything at once
2. **Build adjacent products** — Each product expands TAM and deepens moat
3. **NRR > New logos** — 75% of growth from existing customers
4. **Self-serve is the top of funnel** — Let users prove value before engaging sales
5. **Segmented GTM** — Different motions for different customer sizes
